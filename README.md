# rfTSP

This is an R package for robust binary/multi classification and feature selection in gene expression profiles.

# Required packages
* library(randomForest)
* library(doParallel) 
* registerDoParallel()
* library(foreach) 
* library(truncnorm)

## Install this package from github 
* in R console: 
```R 
library(devtools) 
install_github("KellyCahill/rfTSP") 
```
